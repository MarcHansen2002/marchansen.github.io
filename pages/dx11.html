This project was started for an assignment in university to produce a game using DirectX11 in C++. I decide to create mini-golf as I wanted to try make my own physics which went surprisingly well and was a fun experience learning about how the physics of balls bouncing off of walls works. In another module we had been taught slightly about using ImGUI to create testing statistics you could view to help with optimisation. I took that knowledge and used ImGUI in this project too so I could produce a level editor that could be used to produce levels for the game.
