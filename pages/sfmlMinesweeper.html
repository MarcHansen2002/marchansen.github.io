I started this project to try and increase my skills and knowledge in using C++ and the structure of game engines by making a game without one. I chose minesweeper for this because it's been a game that I've enjoyed ever since I first learnt how to actually play it in 2013. It also has a game loop that isn't too complicated to recreate to use for learning.
